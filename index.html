Meus Projetos

Projeto Web Standard

Este projeto foi desenvolvido como parte da disciplina Web standards na faculdade e visa a atualização do site da loja Parada das Festas. O objetivo é revitalizar a presença online da loja, que já possui um site, mas necessita de melhorias em sua estrutura e reorganizar as páginas existentes.

Projeto TecBlog

Projeto Angular

Com uso do Angular CLI. O projeto também faz uso do Angular material. Além disso, foram aplicadas técnicas de HTML e CSS para aprimorar o design e tornar o visual mais atrativo e intuitivo

Projeto API

Este projeto é uma aplicação web desenvolvida para gerenciar o agendamento de consultas de forma prática e eficiente. A aplicação utiliza um backend em PHP com uma API RESTful conectada a um banco de dados MySQL, proporcionando operações de CRUD (Criar, Ler, Atualizar, Deletar) para usuários e agendamentos.
